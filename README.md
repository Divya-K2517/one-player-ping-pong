**Description:**  
The T-rex game that shows up when there is no internet  

**Technology:**  
Pygame library  

**Prerequisites/Instructions:**  
To start the game,
* install Python and the pygame library
* run main.py  

**Controls:**  
* press the space bar to make the dino jump  
* if the game ends, press "a" to restart
