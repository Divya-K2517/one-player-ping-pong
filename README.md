**Description:**  
A one player ping pong game

**Technology:**  
Pygame library  

**Prerequisites/Instructions:**  
To start the game,
* install Python and the pygame library
* run main.py  

**Controls:**  
* use your mouse to move the paddle horizontally across the bottom of the screen and hit the ball
